      <li><strong>Game&nbsp;3</strong><br>
        Game 3 starts by creating a new triangle.  
        It then draws the length of the opposite, adjacent, or hypotenuse.  
        <b>The first side shown is chosen at random.</b>   
        On each iteration of the game (three in total), a new triangle is drawn.  
        Each time, the associated angle θ is displayed along with the random side.<br>
        In this case, the opposite side is shown.<br>
        <img width="569" alt="G3‑1"
             src="https://github.com/user-attachments/assets/d03d5d7a-bd0e-46fa-a018-dbec80810a31" /><br>
        The user is prompted to enter the missing two lengths (adjacent and hypotenuse).  
        <b>If the answer is wrong, the program prompts again until it’s correct, or the user skips the problem.</b><br>
        The process then restarts, and the user inputs the two missing lengths until they correctly answer the questions or decide to quit.
